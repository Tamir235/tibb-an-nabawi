# tibb-an-nabawi
Ein evidenzbasiertes interaktives Nachschlagewerk zu den gesundheitsbezogenen Überlieferungen des Propheten Muhammad ﷺ. 25+ authentifizierte Hadithe zu Ernährung, Fasten, Schlaf und Hygiene — jeweils mechanistisch mit moderner Wissenschaft verglichen. Pure HTML/CSS/JS, keine Abhängigkeiten.
